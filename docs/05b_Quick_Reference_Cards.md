## How to use this file

Each card below is a self-contained one-pager. Print at A5. Laminate. Put on the back-bar next to the till and one set in the staff room. When in doubt on a shift, look at the cards before posting. If the cards don't answer your question, the rulebook (`05_Staff_Content_Guidelines.md`) does. If the rulebook doesn't, message Roger.

The cards intentionally repeat information from the rulebook. Repetition is the point.

## Card 1 — Pre-Post Checklist

**Before any post, Story, or Reel goes live, tick all five.**

```
[ ] 1. PILLAR
    Is it Machines, Drinks, Vibe, or Events?
    If it doesn't fit one, don't post it.

[ ] 2. CONSENT
    Any customer face? Got a verbal yes?
    Any minors? Any drunk people? STOP.

[ ] 3. AUDIO
    Machine/room sound, or licensed library track only.
    No background Spotify. Mute if unsure.

[ ] 4. CAPTION
    No banned words. Specific (machine, drink, night).
    Australian English. Sounds like a text, not a press release.

[ ] 5. TAG & LOCATION
    Location: "Beercade Australia, Redfern".
    3-5 hashtags from the approved list. Not 20.
```

Can't tick all five? Don't post yet.

## Card 6 — Lighting & Framing Quick Fix

**The two-rule fix for 90% of bad phone shots in this room.**

```
RULE 1   Light source BEHIND YOU, not behind the subject.
         If the neon is behind the cocktail, the cocktail is
         a silhouette. Walk around it.

RULE 2   Too dark? MOVE CLOSER to a light source.
         Don't pull up exposure in the app. That adds noise.
         Move 30cm. Try again.
```

**Other phone rules:**

- Vertical for Reels, TikTok, Stories. Landscape only for genuine wide subjects.
- Get closer than feels comfortable. Pinball fills the frame. Pint fills the frame.
- Tap to focus on the subject before recording.
- Phone torch: off. Flash: off. Brightness in editing: don't fight Beercade's lighting.
- Talking head: phone a forearm's length away, in the quietest corner.
- Music in the background: shoot silent, add a licensed track in the edit.
